# FitnessHub 🏋️‍♂️💪
### Django Fitness Platform by Team Codex

FitnessHub is a feature-rich, high-performance Django e-commerce platform designed for fitness enthusiasts and retail management. It features a dark-themed storefront, complex inventory organization, built-in order tracking, and an automated background script architecture for seamless project cloning and immediate setup.

---

## 🌟 Core Platform Features

### 🛍️ The 17 Fitness Marketplace Categories
The catalog is split into 17 distinctive niche fitness hubs, complete with responsive layouts and dedicated media pipelines:
1. Cardio Training & Equipment
2. Wearable Tech & Trackers
3. Yoga & Pilates
4. Weight Loss Solutions
5. Muscle & Strength Build
6. Supplements & Nutrition
7. Recovery & Wellness
8. Mobility Training
9. Outdoor Adventure Gear
10. Sleep Optimization Tools
11. Kettlebell Workouts
12. Activewear Clothing
13. Bodyweight Calisthenics
14. Dumbbell Training
15. Resistance Bands
16. Gym Machines (Functional, Selectors, and Heavy Rod Racks)
17. Hydration Systems

### 📦 End-to-End Order Tracking System
* **Live Lifecycle Tracking:** Users can view the real-time fulfillment status of their fitness gear purchases.
* **Status Updates:** Transparent tracking transitions from *Processing Order* ➡️ *Package Assembled* ➡️ *Shipped/In Transit* ➡️ *Delivered*.
* **Account Integration:** Secured histories link users to their unique billing invoices and package routes.

### ⚡ Lightning-Fast Curation Control Panel
Built directly into the admin framework to speed up catalog styling:
* **`Ctrl + V` Hotkey:** Seamlessly paste an image URL or clipboard asset path into an unlinked product item.
* **`H` Hot-Save Key:** Instantly save the progress and jump dynamically to the next blank product item without touching the mouse.

### 🎨 Premium User Interface
* **Custom Animated Brand Logo:** A pulsing CSS-animated barbell icon integrated cleanly into the main sticky navbar.
* **Dynamic Sidebar Filter Grid:** Sidebar filtering widgets allowing real-time category, price range, and stock availability toggles.
